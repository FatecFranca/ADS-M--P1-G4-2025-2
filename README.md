## 📖 Sobre o Projeto

Este projeto foi desenvolvido pelo **Grupo 04 do 1º semestre de Análise e Desenvolvimento de Sistemas (MATUTINO - 2025/2)** com o objetivo de criar uma **Landing Page dedicada ao bazar da Casa de Apoio I.A.N.S.A**.

A Casa de Apoio I.A.N.S.A oferece suporte a pessoas em tratamento de câncer, e durante a análise do site institucional, identificamos que o bazar — uma importante fonte de arrecadação — não possuía o devido destaque.

Dessa forma, foi proposta e desenvolvida uma solução focada em:

- 📈 Aumentar a visibilidade do bazar  
- 💰 Potencializar a arrecadação de recursos  
- 🤝 Fortalecer a rede de apoio  
- 🙋 Atrair novos voluntários
