# ADS-M-P1-G4-2025-2
Grupo 04 do 1o semestre de ADS - MATUTINO 2025 2 - ONG CONNECT 

## Projeto
Esse trabalho tem como objetivo a criação de uma Landing Page para o bazar da Casa de Apoio I.A.N.S.A, que hospedam pessoas em tratamento de câncer. Vimos a necessidade de uma página específica para o bazar, já que a página principal deles não é colocado em evidência as vendas. Com isso, ajudaremos a instituição a melhorar os lucros, visibilidade e uma melhor rede de apoio, com um número maior de voluntários.
